# Power BI Internship Projects

## Internship Company
Thiranex

## Project Overview
This repository contains four Power BI projects completed during the internship. Each project focuses on different aspects of data analytics, business intelligence, and reporting.

---

# Task 1: Sales & Revenue Analysis Dashboard

### Objective
Analyze sales performance and revenue trends.

### Features
- KPI Cards
- Monthly Sales Trend
- Top Products
- Sales by Category
- Profit by Region
- Interactive Filters

### Outcome
Developed a business dashboard for sales performance monitoring.

---

# Task 2: Customer Segmentation Dashboard

### Objective
Segment customers based on purchasing behavior.

### Features
- Customer Segmentation
- Region Analysis
- Category Preference
- Top Customers
- Interactive Dashboard

### Outcome
Analyzed customer behavior and purchasing patterns.

---

# Task 3: Predictive Analytics Dashboard

### Objective
Analyze historical sales data and forecast business trends.

### Features
- Historical Sales Trend
- KPI Cards
- Sales by Region
- Category Analysis
- Trend-Based Forecast Insights

### Outcome
Performed predictive trend analysis using historical data.

---

# Task 4: Data Cleaning & Reporting Automation

### Objective
Automate data cleaning and reporting workflows.

### Features
- Missing Value Check
- Duplicate Removal
- Data Type Validation
- Automated Reporting Dashboard

### Outcome
Created reusable automated reporting workflows using Power Query.
