
# Predictive Analytics Dashboard

## Objective
Analyze historical sales trends and perform predictive analysis using Power BI.

## Tools Used
- Power BI Desktop
- Sample Superstore Dataset

## Features
- Historical Sales Trend
- KPI Cards
- Sales by Region
- Sales by Category
- Top Products Analysis
- Interactive Filters
- Trend-Based Forecast Insights

## Prediction Insight
Historical sales data indicates an overall increasing trend with periodic fluctuations. Based on this trend, future sales are expected to remain stable and continue growing gradually.

## Outcome
Performed trend analysis and created a predictive analytics dashboard for business forecasting.
