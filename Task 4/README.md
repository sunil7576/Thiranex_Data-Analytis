# Data Cleaning & Reporting Automation

## Objective
Clean sales data and automate reporting using Power BI.

## Tools Used
- Power BI Desktop
- Power Query
- Sample Superstore Dataset

## Data Cleaning Steps
- Checked missing values
- Removed duplicate records
- Verified data types
- Cleaned and transformed data

## Reporting Features
- KPI Cards
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Product Report
- Interactive Filters

## Automation
Power Query stores all cleaning steps. Whenever new data is refreshed, the cleaning process is automatically applied.

## Outcome
Created an automated reporting dashboard with reusable data-cleaning workflows.
