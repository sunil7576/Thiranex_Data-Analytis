# Sales & Revenue Analysis Dashboard

## Objective
Build an interactive Power BI dashboard to analyze sales and revenue performance using historical sales data.

## Tools Used
- Power BI Desktop
- Sample Superstore Dataset (CSV)

## Features
- Imported sales data from CSV
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Quantity KPI
- Monthly Sales Trend
- Top Performing Products
- Sales by Category
- Profit by Region
- Interactive Slicers (Date, Category, Region, Segment)

## Key Insights
- West region generated the highest sales.
- Technology category contributed significantly to revenue.
- Sales showed fluctuations across different months.
- Interactive filters allow dynamic analysis.

## Outcome
Developed an interactive dashboard for business performance monitoring and sales analysis.
